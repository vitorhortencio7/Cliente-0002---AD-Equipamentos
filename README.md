# AD Equipamentos CE - Website Oficial

Este é o repositório do website oficial da **AD Equipamentos CE**, uma solução integrada de alta performance desenvolvida para apresentar projetos 3D e produtos de refrigeração comercial e gôndolas para supermercados, padarias, mercadinhos, açougues e comércio geral no Ceará.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando um ecossistema moderno e altamente otimizado para web rápida, responsiva e com excelente ranqueamento orgânico:

- **React 18+** - Biblioteca declarativa altamente modular para construção de interfaces.
- **TypeScript** - Tipagem estática para robustez operacional e prevenção de bugs.
- **Tailwind CSS** - Framework de estilização baseado em classes utilitárias de alta velocidade.
- **Motion (framer-motion)** - Implementação de animações suaves e efeitos sensoriais de rolagem (*Scroll Reveal* / *Scroll Appear*).
- **Lucide-React** - Conjunto unificado de ícones vetoriais modernos e leves.
- **Vite** - Build tool ultrarrápido para desenvolvimento moderno no ecossistema web.

## 🚀 Como Executar o Projeto Localmente

1. **Instalar Dependências:**
   ```bash
   npm install
   ```

2. **Iniciar Servidor de Desenvolvimento:**
   ```bash
   npm run dev
   ```

3. **Verificar Qualidade de Código (Lint):**
   ```bash
   npm run lint
   ```

4. **Compilar para Produção (Build):**
   ```bash
   npm run build
   ```
